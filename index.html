02 — About This Project

Project Overview

This portfolio was created as a final project for Dr. Bobby Bothmann's Advanced Bibliographic Metadata in May 2026. Each MARC 21 record was created following descriptive cataloging standards, spanning a range of works and formats to demonstrate cataloging proficiency.

Standards & Facsimiles

All records follow MARC 21 Format for Bibliographic Data as maintained by the Library of Congress. Descriptive elements adhere to Resource Description and Access (RDA) conventions. Facsimile PDFs are available for download with the link in each record's detail panel.
